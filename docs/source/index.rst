pyIncore-incubator modules
================

**pyIncore-incubator**  is a python package consisting of new analyses that
build on pyIncore, but are not yet part of the core analyses.

To **update** pyIncore-incubator run the following command:

.. code-block:: bash

    conda update -c in-core pyincore-incubator
